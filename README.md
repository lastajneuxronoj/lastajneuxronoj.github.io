# lastajneuxronoj

Generador estático para el blog Lastaj Neŭronoj.

## Estructura

* `posts/` → archivos Markdown de los posts.
* `blog/` → HTML generado.
* `scripts/` → scripts de generación.
* `stats/` → estadísticas e historial editorial.
* `data/wordpress/` → datos históricos importados desde WordPress.
* `analytics/` → análisis y dashboard en R.

## Scripts principales

* `build-posts.js` → genera el sitio.
* `build-sitemap.js` → genera `sitemap.xml`.
* `build-stats.js` → genera estadísticas e historial.
